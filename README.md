# enrollment

## About this gem
 이 패키지는 모든 신청에 대한 프론트와 백엔드를 구현하는 것을 목표로 합니다. 이 패키지는 다음의 사례에 유용하게 쓰일 것입니다.
 - 수강신청 시스템이 필요한 프로젝트
 - 물품 구매 시스템을 만들어야 하는 프로젝트
 - 주기적으로 받아야 하는 예약 시스템

 이를 위해서는 다음의 기능이 필요합니다.
 - 사용자 페이지
  - 신청 가능한 목록을 표시
  - 항목에 들어가면 이미지와 정보 출력
 - 관리자 페이지
  - 카테고리/목록 관리
  - 신청 목록 만들기
    - 마감 조건 설정: 기간 설정, 수량
    - 이미지와 정보 입력, 생성
    - CRUD
